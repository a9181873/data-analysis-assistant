> v3.0 | 全本機 AI 數據分析平台

## 產品亮點

### 完全離線、資料零外洩
所有 AI 推理、統計分析、機器學習都在**你的電腦**上運行。資料不經過任何雲端服務。適合處理敏感的金融、醫療、企業內部數據。

### 風控建模一站完成
從 **WOE/IV 變數篩選** → **模型訓練** → **KS/Lift/Gain 評估** → **SHAP 解釋** → **模型匯出** → **PSI 上線監控**，完整覆蓋信用風險建模全流程。

### Docker 一鍵移植
`docker-compose up` 即可在任何電腦啟動，無需手動安裝 Python 或依賴。團隊共享、跨機器部署零障礙。

### AI 助手 + 知識庫
上傳公司內部文件到 RAG 知識庫，AI 助手會自動引用相關資料回答問題。結合 LangChain ReAct Agent，支援自然語言驅動的數據分析。

## 5 分鐘快速上手

### 第 1 步：安裝 Ollama
前往 [ollama.ai](https://ollama.ai/) 下載並安裝

### 第 2 步：下載模型
```bash
ollama pull qwen2.5:14b    # 32GB RAM 推薦
# 或
ollama pull qwen2.5:7b     # 16GB RAM
```

### 第 3 步：安裝依賴

**方法 A — 使用安裝腳本：**
- Windows: 雙擊 `install.bat`
- Mac/Linux: `./install.sh`

**方法 B — 手動安裝：**
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# Mac/Linux: source .venv/bin/activate
pip install -r requirements.txt
```

### 第 4 步：啟動
```bash
streamlit run streamlit_app.py
```

瀏覽器自動開啟 `http://localhost:8501`

### 第 5 步：開始使用
1. 上傳數據檔案（CSV、Excel 等）
2. 八個分頁，依序探索
3. 用「AI 助手」自然語言提問

## Docker 一鍵部署（移植推薦）

```bash
# 1. 安裝 Docker Desktop (docker.com)
# 2. 執行：
docker-compose up --build
# 3. 開啟 http://localhost:8501
```

移植到其他電腦只需：**複製資料夾** → **安裝 Docker** → **docker-compose up**
